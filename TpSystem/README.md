# TPSystem --BDS传送系统

## 1. 安装
> 请确保你的服务器已安装并运行Liteloader   

    1. 下载**TPSystem.js**  
    2. 扔到万能的**Plugins**文件夹  
    3. 启动或重启服务端，待Liteloader 输出JS插件TPSystem已加载，则代表安装成功
